## ğŸ” Troubleshooting

### Issue: Progress bar not showing
**Solution**:
```bash
# Check if tqdm is installed
pip show tqdm

# Install if missing
pip install tqdm>=4.66.0
```

### Issue: Cache not working
**Solution**:
```python
# Check cache location
from src.utils.hash_cache import get_hash_cache
cache = get_hash_cache()
print(f"Cache DB: {cache.db_path}")

# Clear cache if corrupted
cache.clear_old_entries(days=0)
```

### Issue: Worker count not adapting
**Solution**:
```python
# Verify adaptive_workers is enabled
converter = DocumentConverter(adaptive_workers=True)

# Check file categorization
categorized = converter._categorize_by_size(files)
print(categorized)
```

### Issue: Performance not improving
**Checklist**:
- âœ… Run conversion at least twice (first run builds cache)
- âœ… Verify `logs/hash_cache.db` exists
- âœ… Check that files haven't changed between runs
- âœ… Ensure `adaptive_workers=True`
